What's the Mac news?  Are you up and running with a new machine?  If
so, here are some tools you'll need to install:

  Java (no longer installed by default;  http://support.apple.com/kb/DL1572)
     You want both Java 6 and 7 jdks installed.  For 7:
http://www.oracle.com/technetwork/java/javase/downloads/index.html

  XCode from apple.  You have to register as a developer and download
from the App Store.
  iTerm2 http://www.iterm2.com/#/section/home
  Ukelele http://scripts.sil.org/cms/scripts/page.php?site_id=nrsi&id=ukelele
  (also see https://coldie.net/?p=103)
  Homebrew http://mxcl.github.com/homebrew/
  Bash 4  (install using homebrew:  $ brew install bash)
  emacs for macs http://emacsformacosx.com/ (not aquamacs)
  mactex  http://tug.org/mactex/
  leiningen http://leiningen.org/
  clojure http://clojure.org/
  skim  http://skim-app.sourceforge.net/
  jetty  http://jetty.codehaus.org/jetty/

Various Java XML/RDF tools.  (I keep my jar files in /usr/local/jar,
but ~/.jar or anywhere else would do as well).

   Jing  http://www.thaiopensource.com/relaxng/jing.html
   Trang  http://www.thaiopensource.com/relaxng/trang.html
  RDF2RDF  http://www.l3s.de/~minack/rdf2rdf/
  FOP  http://xmlgraphics.apache.org/fop/  or  XEP
http://www.renderx.com/tools/xep.html
  Jena Eyeball
http://jena.apache.org/documentation/tools/eyeball-getting-started.html
  Apache Commons Resolver
http://projects.apache.org/projects/xml_commons_resolver.html
  Xalan  http://projects.apache.org/projects/xalan_for_java_xslt_processor.html
  Xerxes http://projects.apache.org/projects/xerces_for_java_xml_parser.html

There are also various bits of sysadmin you need to do such as setting
up an admin account and sudo support, let me know when you're ready
and I'll talk you through it.  See also
https://discussions.apple.com/thread/3128749,
http://humandecisionrequired.com/?p=76,
http://superuser.com/questions/293734/on-mac-how-do-i-give-myself-permanent-sudo-privleges

Always logon to a standard user account, not an admin account.

Cheers,
